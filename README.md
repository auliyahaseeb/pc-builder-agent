The Master Architect: Autonomous PC Systems Engineer
The Master Architect is a specialized, constraint-aware AI Agent designed to disrupt the traditional PC building experience. Unlike generic LLMs that rely on static, often outdated training data, this agent utilizes real-time web-scraping and a deterministic hardware validation engine to generate 100% compatible, price-optimized hardware configurations.  

Quick Start
Environment: Copy .env.example to .env and add your OPENAI_API_KEY.

Install: pip install -r requirements.txt

Run Simulation: python -m src.main

Test: pytest tests/
